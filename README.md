# Spontaneo (Console App)

Spontaneo (Console App Version) is a program that helps users find spontaneous activities based on their preferences.

The program asks the user for:
- The city they are in
- Their budget
- Whether they want an indoor or outdoor activity
- The vibe of the trip (chill or adventurous)

Based on these inputs, the program selects a random activity that matches the user's preferences. It also generates a **Spontaneo Comfort Zone Challenge** to encourage users to try something new and step outside their comfort zone.
